# 📊 Customer Behavior Analysis Dashboard

## 📌 Project Overview

This project analyzes customer purchasing behavior to identify buying patterns, customer segments, and product preferences. The data was cleaned and transformed using Python, analyzed with SQL, and visualized through an interactive Power BI dashboard to generate actionable business insights.

---

## 🎯 Objectives

- Analyze customer purchasing behavior.
- Identify top-selling products and categories.
- Understand customer demographics and spending patterns.
- Build an interactive dashboard for business decision-making.

---

## 🛠️ Tech Stack

- **Python**
  - Pandas
  - NumPy
  - Matplotlib
  - Seaborn
- **MySQL**
- **Power BI**
- **Jupyter Notebook**

---

## 📂 Project Workflow

1. Data Collection
2. Data Cleaning & Preprocessing (Python)
3. Exploratory Data Analysis (EDA)
4. Data Loading into MySQL
5. SQL Analysis
6. Dashboard Development in Power BI

---

## 📊 Dashboard Features

- Total Customers
- Total Sales
- Category-wise Sales Analysis
- Top Purchased Products
- Customer Demographics
- Interactive Filters & Slicers

---

## 🔍 Key Insights

- Identified the highest-selling product categories.
- Discovered customer purchasing trends across different age groups.
- Compared spending behavior by gender.
- Identified high-value customer segments.
- Generated insights to support marketing and inventory planning.
